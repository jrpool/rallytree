RallyTree introduction

Trees

Do you need to operate on a tree of Rally work items?

To do that manually, you must

  1. expand each work item in the Rally web interface, until the entire tree is expanded
  2. perform a bulk (if possible) or item-by-item edit

Trees of Rally work items can have hundreds of nodes. One operation can take minutes or even hours. It is also vulnerable to error.

RallyTree automates that process. You specify the root user story of the tree and what you want done. Then RallyTree performs the operation on the tree.

Operations

With RallyTree, you can:

  • Create a copy of, or a test plan from, a tree
  • Change properties and affiliations of work items in a tree, including owner, project, release, iteration, schedule state, state, test folder, and test set
  • Add items to a tree, including tasks, test cases, and test results
  • Document a tree
